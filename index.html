Bubble Sort

Bubble Sort repeatedly compares two adjacent elements and swaps them if they are in the wrong order. It keeps doing this until the array is fully sorted. Simple, but slow for large arrays.

Time Complexity O(n²)
Space Complexity O(1)
Stable Yes
